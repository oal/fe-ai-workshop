# AI Coding Workshop for Frontend Developers

En praktisk workshop om hvordan bruke AI-verktøy som GitHub Copilot og Claude for mer effektiv frontend-utvikling.

## Om workshopen

Denne workshopen tar for seg hvordan frontend-utviklere kan bruke AI som en planleggingspartner og debugging-kollega - ikke bare som en autocomplete-funksjon. Gjennom praktiske øvelser lærer du å:

- Bruke AI for å planlegge arkitektur og løsninger
- Få hjelp til debugging og feilsøking
- Generere kode fra design
- Jobbe mer effektivt med HTML, CSS, JavaScript og React

**Varighet:** 1,5 timer

**Målgruppe:** Frontend-utviklere med erfaring i HTML, CSS, JavaScript og TypeScript

## Forutsetninger

Før du starter workshopen, må du ha følgende installert og konfigurert:

- **Node.js 20+** - For å kjøre eksemplene
- **VS Code** - Med GitHub Copilot-extension
- **GitHub Copilot-abonnement** - For AI-assistert koding
- **Anthropic-konto** - For tilgang til Claude

Se [SETUP.md](./SETUP.md) for detaljert installasjonsveiledning.

## Øvelser

Workshopen består av 4 praktiske øvelser:

1. **Warm-up (HTML/CSS/JS)** - Kom i gang med AI-assistert koding av grunnleggende web-komponenter
2. **React App Enhancement** - Utvid en eksisterende React-applikasjon med nye features
3. **Debug Challenge** - Bruk AI til å finne og fikse bugs i eksisterende kode
4. **Design-to-Code** - Generer kode fra designfiler og mockups
