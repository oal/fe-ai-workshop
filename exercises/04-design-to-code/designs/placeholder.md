# Design Assets

## For Track B Students (Screenshot-based approach)

This folder contains design assets for Exercise 4.

---

## Included: Material UI Dashboard Mockup

We have included a ready-to-use dashboard mockup:

**File:** `dashboard-mockup.html`

**How to use:**
1. Open `dashboard-mockup.html` in your browser (double-click or drag to browser)
2. Take a screenshot (Win + Shift + S on Windows, Cmd + Shift + 4 on Mac)
3. Save as `dashboard.png` in this folder
4. Use this screenshot in VS Code Copilot Chat or Claude Code

**Alternative:** Reference the HTML file directly in your prompt:
```
"Read the file exercises/04-design-to-code/designs/dashboard-mockup.html and implement it as a React component with TypeScript"
```

## Getting More Designs

You have several options for finding additional designs:

### Option 1: Use Online Dashboard Examples
Search for and screenshot dashboard designs from these sources:

**Material UI Dashboards:**
- Search for "Material UI dashboard" on Google Images
- Visit [Material UI Templates](https://mui.com/material-ui/getting-started/templates/)
- Look for free dashboard templates at [Creative Tim](https://www.creative-tim.com/templates/free)

**React Admin Dashboards:**
- Search for "React admin dashboard" or "React dashboard template"
- Check out [Ant Design Pro](https://pro.ant.design/) for inspiration

**General Design Inspiration:**
- [Dribbble](https://dribbble.com/) - Search for "dashboard UI" or "admin panel"
- [Behance](https://www.behance.net/) - Search for "web component design"

### Option 2: Screenshot from Existing Websites
Find a component or layout you like on any website and take a screenshot:

1. Open the website in your browser
2. Use Windows Snipping Tool (Win + Shift + S) or similar
3. Save the screenshot to this `designs/` folder
